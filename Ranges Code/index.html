Organized Chaos

Inspired by Mid-century modern design, Organized Chaos uses basic geometric shapes
as building blocks to create the individual identities of each character. A collage effect
is implemented, creating a juxtaposition between the delicacy of the thin, overlapping
outlines and the overall heaviness of each letter. Elegant yet fun, this typeface makes
for a great display type, adding a touch of intrigue to any body text.
